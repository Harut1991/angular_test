SUBMIT A CODE SAMPLE

The initial stage consists of a few quick steps that will take close to no time from you. This is what you need to do:

  • Sign up on Kodeist: it’s incredibly easy and fast. Sign in if you are already registered.
  • Now go to your Dashboard and click on +Add sample button so as to submit your code sample.
  • This will redirect you to an illustrated page where you will be asked to fill in a few fields and submit the code. The following are obligatory: Unique name of the code sample, repository link (in this case there will be no requirement for a zip file), short description of the code, and last but not least, the focus of the code checking process i.e. what you want to have reviewed exactly.

Note: You will be able to change or edit the code after its submission. Make sure you do that when the status of the code is “Pending” as you won’t have this functionality in case of a “In Review” code status.

CODE REVIEW

Now that you have submitted your code sample, it’s high time to proceed to the second stage of your code review process which will include the following actions:

  • Here is when our highly qualified developers come to the fore to review your code. A code review sample will be sent to you.
  • When the developers are done with the reviewing of your code, you can find it in the Completed section on the dashboard.
  • Feel free to download the file in .pdf and look up the issues detected in your code.

Dashboard

It’s also important to mention the way the dashboard should be used as part of this overall process. The dashboard enables you to manage all your code samples which you have submitted. By manage we mean the following functionalities and more:

  • Editing
  • Changing the deadlines
  • Renaming
  • Deletion
  • Seeing the status

The dashboard is made up of 3 main filters which are:

  • Pending reviews (this includes the ones that have just been submitted and have not been started yet),
  • On review (the ones in the reviewing process)
  • Completed reviews (the code samples which have already been reviewed)

PAYMENT

At the 3rd stage you will be required to make the payment.
